
## Описание
[GitHub - Alintor/obsidian-kinopoisk-plugin: Obsidian Kinopoisk plugin](https://github.com/Alintor/obsidian-kinopoisk-plugin#example-template)

Поиск информации о фильмах и сериалах в Кинопоиске
Используйте API Kinopoisk.dev для получения информации.

## Определения переменных шаблона
Здесь вы найдете определение возможных переменных, которые будут использоваться в вашем шаблоне. 
Просто напишите `{{name}}`в вашем шаблоне и замените имя нужными данными.
==Важно!== Если указываете данные параметры в свойствах файла то нужно еще добавлять
`""` что-бы получилось:  `"{{name}}"` иначе шаблон будет съезжать при любой правке свойств

| имя                 | описание                                                   |
| ------------------- | ---------------------------------------------------------- |
| name                | Название фильма/телепередачи.                              |
| alternativeName     | Альтернативное название фильма/сериала                     |
| year                | Год создания фильма/сериала                                |
| countries           | Страны, участвовавшие в разработке этого фильма/телешоу    |
| countriesLinks      | Список стран встроен в ссылку Obsidian [[]]                |
| genres              | Жанры этого фильма/сериала                                 |
| genresLinks         | Жанры этого фильма/сериала встроены в ссылку Obsidian [[]] |
| director            | Имя режиссера фильма.                                      |
| movieLength         | Продолжительность фильма в минутах                         |
| posterUrl           | URL-адрес постера фильма/телешоу.                          |
| description         | Описание фильма/сериала.                                   |
| kinopoiskUrl        | URL страницы фильма/сериала на Кинопоиске                  |
| seasonsCount        | Количество сезонов в сериале                               |
| seriesInSeasonCount | Среднее количество серий в сезоне телешоу                  |
| seriesLength        | Средняя продолжительность серии телешоу в минутах          |
| isComplete          | Завершено телешоу или нет (может быть правдой или ложью)   |
| ratingKp            | Пользовательский рейтинг на КиноПоиске                     |
| ratingImdb          | Пользовательский рейтинг на IMDB                           |
| imdbId              | Идентификатор фильма/телешоу на IMDB                       |
| tmdbId              | Идентификатор фильма/телешоу в TMDB                        |
| kpHDId              | ID фильма/сериала на КиноПоиск HD                          |
