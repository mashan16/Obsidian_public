Настройка NTP Debian (Настройка времени)
Вывод информации о времени
`timedatectl status`
Установить часовой пояс Иркутск
`timedatectl set-timezone Asia/Irkutsk`

Если после смены часового пояса команда `date` показывает в 12 часовом формате AM/PM
Проверяем локаль? 
```
locale
```
если локаль `en_US.UTF-8` то она использует 12-часовой формат времени (AM/PM)
Что бы поменять на 24 формат меняем переменную в `~/.bashrc`
```
export LC_TIME=C.UTF-8
```
