Статья [sysadmin.pm](https://sysadmin.pm/httping/)

httping — простая утилита, с помощью которой можно «пинговать» сайт по определённому адресу и получать общую (а при необходимости подробную) статистику по ответу и доступности.

Устанавливаем
```
apt install httping
```

Для запуска пинга выполняем:

```
httping -Y -c4 -s https://sysadmin.pm/
```

Здесь -Y активирует вывод в цвете, -с указывает на количество соединений, а -s отображает код ответа от сервера.

![httping](https://sysadmin.pm/wp-content/uploads/2018/02/httping.png)

Дополнительных опций у утилиты очень много — поддерживается подробный вывод с разделением на составляющие (резолв, подключение, отправка и т. д.). Имеется возможность управлять настройками сети — задать MTU, указать размеры буферов, забиндить утилиту на определённый IP и многое другое. Поддержка аутентификации, SSL, IPv6 есть из коробки.

Доступен даже простой GUI для отображения статистики, вызывается он так:

```
httping -c100 -K https://sysadmin.pm/
```

![httping](https://sysadmin.pm/wp-content/uploads/2018/02/httpingui.png)

Полный список команд и опций к ним можно получить выполнив:

```
# httping --help
```

Утилита так же доступна и для Android смартфонов. Поставить её туда можно из Play Market’а.
